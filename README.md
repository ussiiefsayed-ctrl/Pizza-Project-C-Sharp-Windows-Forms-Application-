# Pizza-Project-C-Sharp-Windows-Forms-Application-
A Windows Forms Pizza Ordering System built using C#. The application allows users to customize their pizza by selecting size, crust type, toppings, and dining preference, with real-time price calculation.

The project uses structured event-driven programming to handle user interactions across multiple forms (Main Menu, Order Menu, and Final Bill). It maintains an order summary using structs and dynamically updates the UI based on user selections.

The system demonstrates strong understanding of Windows Forms development, form communication, state management, and clean separation between UI logic and business logic.
